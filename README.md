# Password-security-analyzer
Password Strength Analyzer Hackathon Project

## Problem
Many users create weak passwords that are vulnerable to brute-force attacks.

## Solution
This web application analyzes password strength in real-time and provides improvement suggestions.

## Features
- Real-time password strength detection
- Strength meter visualization
- Security suggestions
- Crack time estimation
- Password generator

## Technologies Used
- HTML
- CSS
- JavaScript

## Hackathon Project
Developed as part of a cybersecurity hackathon.
